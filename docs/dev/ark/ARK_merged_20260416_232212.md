65: ## 1. Project Overview
66: 
67: **ZenSynora (MyClaw)** is a personal AI agent featuring:
68: - **Flexible LLM Providers**: Ollama, OpenAI, Anthropic, Gemini, Groq, etc.
69: - **SQLite-backed Persistent Memory**: Per-user isolation
70: - **Multi-agent Support**: Named agents with delegation capabilities
71: - **Agent Swarms**: Parallel/sequential/hierarchical/voting task execution
72: - **Knowledge Base**: Full-text search (FTS5) with Markdown storage
73: - **Channel Integrations**: Telegram, WhatsApp, CLI
74: - **Task Scheduling**: Via Telegram `/remind` and WhatsApp commands
75: - **Dynamic Tool Building**: Agent can create runtime tools
